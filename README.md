# 🌕 Lunar Crater Detection

Automated detection of impact craters on the Moon's surface 
using deep learning and computer vision.

## Project Overview
Training an object detection model (YOLOv8) on real NASA LRO 
imagery to automatically identify and localize lunar craters.

## Dataset
- **Labels:** LROC Crater Catalogue (19,335 craters)
- **Imagery:** NASA Lunar Reconnaissance Orbiter (LRO) WAC imagery
- **Clean dataset:** 14,914 standard-quality crater annotations
